Work experience

Crédit Mutuel Arkéa

Internship June-July 2013

Part-time internship October 2013-September 2014

"Analysis and implementation of technical metrics on VMware virtualized environments and Windows servers"

I worked as a junior DevOps (developer and sysadmin) at the Systems team at the headquarters of Crédit Mutuel Arkea.

Arkea Realtime Monitoring Infrastructure (Artimon for short) is involving technologies like Hadoop, Flume, Kafka, Zookeeper, or ElasticSearch with in-house Web apps based on D3js.

My job was to add Windows's based servers to Artimon and improve data vizualization.

These are some of my work:

Dev:

  • I merged and upgraded Arkea's version of Apache Flume by recompiling it with Maven and making it available to Windows too.
  • I wrote some VBScripts/PowerShell scripts in order to generate metrics from Windows (using PowerCLI API for VMware vSphere)
  • I enhanced the vizualization apps written with AngularJS and D3JS
  • I created a Restful Web Service to brings data stored in Hadoop to Arkea's vizualization apps (backend written in Java)
  • I developped a Kafka consumer in Java to feed OpenTSDB over HTTP

Ops:

  • I put into production NXLog and Flume for Windows
  • I configured NXLog to bring Eventlogs from Windows to ElasticSearch
  • I created a small Hadoop cluster for some experiments about ElasticSearch for Hadoop, OpenTSDB, and HBase
  • I had an in-house training about Hadoop, Pig and Chef

I also heard a lot about mainframes, OpenStack, CoreOS, Docker, Redis, Nagios, VMware vSphere, Citrix, HAProxy, Varnish and so on...
