    ISO generator for KVM

    Exherbo is a source-based distribution. I wanted to play with it without messing up my laptop(and I'm used to develop through ssh). So I created a bash script that is creating a KVM-based ISO of Exherbo to boot it on OpenStack. This script is including:

    • Fetching exherbo tarball
    • Creating a single btrfs partition
    • Compile latest kernel with KVM options
    • Generate initramfs
    • COnfiguring sshd and other stuffs
    • Adding my ssh key to authorized hosts
    • Generate a ISO of all the above
